# Galería de Portfolios
Sitio web para publicar currículums y portfolios personales.


## Tecnologías Utilizadas
+ **Framework:** django v5.0.1
+ **Código:** Python v3.12
+ **Base de Datos:** SQLite 
+ **Repositorio:** GitHub
+ **Diseño:** CSS with Boostrap v5 

## Componentes del **Equipazo**
+ Álvaro
+ Noa
+ Gema
